## 📲 Installation

1. Télécharge le dernier **APK** via le bouton ci-dessus (ou dans l'onglet [Releases](https://github.com/Gatsby4444/sonlite-app/releases)).
2. Sur ton téléphone, autorise l'installation depuis cette source si Android le demande.
3. Ouvre l'app et profite 🎶

> Android uniquement.

## ✨ Fonctionnalités

- 🎧 **Téléchargement & écoute hors-ligne** — récupère tes sons et écoute-les partout, sans connexion.
- 📂 **Bibliothèque & playlists** — organise, recherche, édite (découpe) tes morceaux.
- 👥 **Social** — profils, abonnements, amis, messagerie, « sons du jour » éphémères.
- ⭐ **Premium** — skins, icônes et **badges de profil** exclusifs pour les membres qui soutiennent le projet.
